# 🚀 Satellite Collision Detection System

An AI-inspired system to predict and prevent satellite collisions using real-time simulation and visualization.

---

## 🌍 Problem

Space is becoming increasingly congested with satellites and debris, leading to a high risk of collisions.

---

## 💡 Solution

This project simulates satellite motion and predicts potential collisions, suggesting preventive maneuvers.

---

## ⚙️ Features

- 🛰️ Real-time satellite orbit visualization  
- ⚠️ Collision detection system  
- 📊 Risk prediction (HIGH / LOW)  
- 🧠 Basic maneuver suggestion  
- 🎮 Interactive user inputs  

---

## 🧪 Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js (Express)  
- Deployment: Vercel (Frontend), Render (Backend)
